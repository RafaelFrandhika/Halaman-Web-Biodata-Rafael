Halaman Tentang Hobi Saya

Saya memiliki beberapa hobi yang saya tekuni: desain grafis, bermain bola, menonton film dan Anime, mendengarkan musik, dan bermain game.

1. Desain

Saya mulai suka mendesain pada kelas 10 semester 1 saat saya sedang mencoba-coba mendesain, mulai dari sana saya memiliki minat desain grafis. Dan saya mencoba lebih rutin lagi dalam mendesain agar dapat mengasah skill saya untuk bekerja.

2. Bermain bola

Mungkin saya sudah tidak terlalu aktif dalam bermain bola dan hanya menonton pertandingan saja, namun sebelumnya saya sering bermain bola karena pengaruh dari Ayah dan Kakak saya yang sangat suka menonton sepak bola. Waktu SD saya pernah ikut turnamen 2 kali meskipun tidak menang, saya sangat menikmati pengalaman tersebut.

3. Menonton Film dan Anime

Saya mungkin masih kurang dalam menonton Film, tapi saya sudah lumayan banyak menonton Anime selama hampir 5 tahun. Awal mula saya menonton Anime adalah pada saat pandemi virus korona, saat itu saya menonton tv rtv muncul series My Hero Academia dan disitulah saya mulai menonton Anime sampai sekarang. Dan top 3 Anime fav saya adalah: 1. Violet Evergarden (10/10) 2. Frieren: Beyond Journey's End (9.8) 3. 86 Eighty-Six (9.5)

4. Mendengarkan Musik

Saya mendengarkan musik biasanya saat sedang mengerjakan tugas dan saat sedang luang, Genre musik fav saya adalah Rap dan J-pop. Artis Rap fav: Kendrick Lamar, Travis Scott Artis J-pop fav: Kenshi Yonezu

5. Bermain Game

Saya sekarang sudah berkurang untuk bermain game tidak seperti dulu, mungkin karena banyaknya tugas atau beban dan munculnya hobi yang lebih produktif. Saya sekarang bermain game kalau ada teman sama gabut saja, dan tidak terlalu saya anggap serius.
